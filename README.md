# API-Tasks

## Description

API to manage Tasks made in Node.Js with Nest.Js, Prisma and Postgres.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# database orchestration (Docker-compose)
$ docker-compose up -d

# development
$ yarn run start
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Documentation

Development Swagger Documentation route: <b>/api</b>
